# uConsole Claude Buddy

Physisches BLE-Companion-/Buddy-Terminal, das auf Claude-Code-Sessions reagiert (Mood, Live-Feed, Sound, Freigaben).

- **`bridge/`** — Mac-Seite: Claude-Code-Hooks + BLE-Daemon (Central).
- **`device/`** — uConsole (Raspberry Pi CM4): Textual-Kiosk-UI + BLE-Peripheral.
- **`docs/`** — Specs + Umsetzungspläne.

Historie beider Teile via git-subtree erhalten.
